//----------------------------------------------------------
// 概要
//----------------------------------------------------------
■回転する球体モデルにモデルを追従させる処理■

■環境■
言語     : C++ 
開発     : DirectX9 Visual Studio 2019 (x86)
開発期間 : 2日

//----------------------------------------------------------
// 実装方法
//----------------------------------------------------------
生成時にモデルが球体のクォータニオン、オフセットを保存し、
逆クォータニオンを使用して生成時と現在で回転した分の差分を求め、
回転行列にする、これをオフセットにTransfromCoordする。回転を適応した
オフセットを使い pos = 球体の中心 + rotateOffsetで最終的に追従
することができました。スケールを適応した際には回転行列にスケール
の情報を入れ、オフセットにTransfromCoordすることで拡大縮小にも
対応できました。
